# スイーツパニック 実装メモ

## 2026-05-27 実装範囲

フル仕様計画をもとに、既存DX11版へ以下を追加した。

- `CharacterType`, `EnemyType`, `BossType`, `StageType`, `PickupType` の分類
- 最大4人枠。現在の通常起動は1Pのみ有効で、AIなしのソロプレイとして開始する
- ダウン、近接救助、全員ダウン時ゲームオーバー
- 短押し通常攻撃、長押し解除チャージ攻撃
- 4キャラの通常/チャージ/必殺の差別化
- 反射倍率、反射キル、鏡敵、チーズ壁の敵弾反射
- 8ステージタイプ、8敵タイプ、6ボスタイプ、10ピックアップ効果
- フィーバー、簡易コア強化、ボス撃破時のキャラ別強化
- 近接している2人が必殺ゲージ満タンの時に発動する合体必殺の実装口
- 4人分HUD、ステージ名、ボス名、グレイズ、ボム、反射キル表示
- WICによるPNG/JPEG/BMP読込、スプライト登録、仮PNGアセット
- `sprite_unlit.hlsl` と `effect_additive.hlsl` の追加

## 残す拡張余地

- FBX/glTFの実読込は未実装。`AssetCatalog::modelPath` と `ModelLibrary` の差し替え口は維持している。
- スプライト描画シェーダーは追加済みだが、現時点の戦闘表示はプリミティブメッシュ中心。
- 20種の個別固有スキル選択UIは未実装。現時点ではキャラ別の簡易強化として扱う。
- P2-P4、AI、救助、合体必殺のコードは残しているが、通常起動ではP2-P4を非アクティブにしている。
